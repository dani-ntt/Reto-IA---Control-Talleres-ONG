# Memory Bank Inicializada

Se ha creado la estructura base de la Memory Bank en `memory-bank/`, empezando por `projectbrief.md`, completamente alineado con el proyecto actual de **Aplicación de Inscripción a Talleres**.

## Estructura prevista

La carpeta `memory-bank/` contendrá como mínimo:

1. `projectbrief.md` (YA CREADO)
   - Fuente de verdad de objetivos, alcance y criterios de éxito.

2. `productContext.md`
   - Por qué existe el producto, qué problema resuelve, escenarios de uso.

3. `systemPatterns.md`
   - Arquitectura backend/frontend.
   - Patrones de diseño y organización de código.
   - Flujo de datos entre capas.

4. `techContext.md`
   - Stack tecnológico concreto (React, CRA, Node, Express, JWT, Jest, etc.).
   - Configuración relevante de desarrollo.

5. `activeContext.md`
   - Qué partes están en foco actualmente (p.ej., tests unitarios frontend).
   - Decisiones recientes.
   - Cosas a vigilar.

6. `progress.md`
   - Qué está terminado.
   - Qué queda por hacer.
   - Estado de pruebas, bugs conocidos.
